# Exercices React Native

### Objectives

Realization of several react native exercises with expo

### Prerequisites

What you need to use it:
  * Node.js | npm

### Installation

Step to install:
  1. ``` git clone https://github.com/capliez/capliez_alexis_M1_exercices_react_native.git ```
  2. ``` npm install ```
  3. ``` expo start ```
